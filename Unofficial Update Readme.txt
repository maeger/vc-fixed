-DESCRIPTION-
This collection of mods aims to fix the game's bugs and refresh the old ui. With this update you can play the game with higher frame rates and higher resolutions. You can also add another mods with the mod loader. I tried to stay as faithful to the original game as possible.

-HOW TO INSTALL-
1. You need a clean install of GTA Vice City.
2. Just extract all of the files from "GTAVC_QoL_Update_V1.0.rar" to your game directory(Example: "C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto Vice City").
3. Play the game from "gta-vc.exe" file.

-INFO-
- To use the radar zoom press T key while playing the game.
- To turn off the radio press X key when you are in a vehicle.
